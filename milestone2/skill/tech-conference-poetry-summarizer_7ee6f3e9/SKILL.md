# tech-conference-poetry-summarizer

## 何时使用
1. 收集技术会议信息（时间、地点、主题、议程）
2. 提取关键技术领域和热点话题
3. 识别产学结合趋势和竞赛活动
4. 按照古典诗歌格式组织内容
5. 添加技术要点注释和背景说明

## 技能内容
# tech-conference-poetry-summarizer

## 技能描述
将技术会议动态转化为诗歌形式的总结技能，适用于快速传达学术会议核心内容和前沿趋势

## 核心指令
采用七言古诗体与现代技术术语结合的方式，将技术会议信息结构化呈现：序章（会议概况）→ 技术章节（核心领域）→ 创新章节（竞赛/趋势）→ 生态章节（产学合作）→ 未来展望 → 技术注释。确保诗歌韵律与技术准确性平衡，每章节4-8句，重点突出会议使命、技术突破和社区价值。

## 使用示例
输入：MLSys 2026会议信息
输出：包含序章、技术浪潮、前沿探索、产学共舞、未来展望等章节的诗歌，附带技术要点注释

## 原始提交内容（参考）
任务完成！我已经创作了一首关于MLSYS（机器学习系统）近况的诗歌，并提交了解决方案。诗歌涵盖了MLSys 2026会议的最新动态、技术竞赛、产学结合趋势以及机器学习系统领域的前沿发展方向。
