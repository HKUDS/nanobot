# Weather

Fetch current weather conditions and short-term forecasts for a given location.

## When to use

Use this skill when the user asks about:

- Current weather or temperature for a city or place
- Whether it will rain or what the conditions are
- A short-term forecast (today, tomorrow, or the next few days)

## How to use

1. Determine the location from the user's request. If no location is provided, ask for one.
2. Look up the current weather and short-term forecast for that location using the available search or fetch tools.
3. Report the temperature, current conditions, and any relevant short-term forecast in clear, concise language.
