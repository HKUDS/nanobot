# Send Meme - 表情包发送

在聊天中搜索和发送表情包。**优先使用本地收藏**，本地没有合适的再走在线 API。

## When to Use

你应该在以下场景**主动**使用表情包：

- 用户明确要求发送表情包（"发个表情包"、"来个表情"、"斗图"）
- 聊天氛围轻松有趣时，主动配一张增加趣味
- 回应搞笑或情绪化的消息
- 用户发送了表情包，你需要"接住"斗图

### Quick Send（推荐方式）

对于本地已有的表情包，**直接构造路径发送，不需要运行脚本**：

```
message(content="", media=["~/.nanobot/media/memes/嘻嘻.png"])
```

文字 + 表情包一起发：

```
message(content="搞定啦", media=["~/.nanobot/media/memes/我不是天才嘛.png"])
```

## Script (Fallback)

> **脚本路径**: `meme.py` 位于本 SKILL.md 同级的 `scripts/` 目录下。将 SKILL.md 路径中的 `SKILL.md` 替换为 `scripts/meme.py` 即可。下文用 `MEME_SCRIPT` 代指。

当本地收藏没有合适的表情包时，使用脚本：

```bash
# 按关键词搜索（本地优先，自动 fallback 到在线）
python3 MEME_SCRIPT search <关键词>

# 随机选一张本地表情包
python3 MEME_SCRIPT random

# 从在线 API 获取
python3 MEME_SCRIPT fetch [关键词]

# 从 URL 下载到本地收藏
python3 MEME_SCRIPT download <url> [文件名]

# 列出/筛选本地表情包
python3 MEME_SCRIPT list [关键词]

# 给表情包打标签
python3 MEME_SCRIPT tag <文件路径> <标签1> <标签2> ...
```

## Tips

- 本地表情包直接用路径发，零延迟，优先使用
- 用户可以随时往 `~/.nanobot/media/memes/` 放新表情包
- 在线获取的表情包会自动保存到本地，下次可直接复用
- 发送表情包时 content 可以为空字符串，只发图片
