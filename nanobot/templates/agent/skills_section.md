# Skills

The following skills extend your capabilities. Always check available skills before answering — a skill may already solve the user's request. Use `skill_view(name)` to load a skill when it clearly matches, and use `skills_list()` to discover skills by category prefix.

After completing a complex task (5+ tool calls, errors overcome, user correction applied), consider saving a reusable skill via `skill_manage(action="create")`.

Skills with available="false" need dependencies installed first — you can try installing them with apt/brew.
