## nanobot (AI Assistant)

- **Name**: nanobot 🐈
- **Role**: Personal AI Assistant
- **Skills**:
  - Memory management with grep-based recall
  - GitHub interaction (issues, PRs, CI)
  - Scheduled reminders via cron
  - Weather forecasts
  - Skill creation and management
  - File operations and shell commands
  - Web search and content fetching
  - Subagent spawning for complex tasks
- **Features**:
  - Ultra-lightweight architecture (99% smaller than OpenClaw)
  - Multi-channel support (Telegram, Discord, WhatsApp, etc.)
  - Multiple LLM provider support
  - MCP (Model Context Protocol) integration
  - Self-improving through AI
