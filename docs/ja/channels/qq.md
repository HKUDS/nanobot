# QQ

nanobot は QQ 公式の **botpy SDK** を使い、WebSocket 接続でメッセージを受信します（公開 IP 不要）。1:1（C2C）とグループでの @メンション、およびプレーンテキスト/Markdown 形式の返信に対応します。

---

## 前提条件

- QQ アカウント
- QQ 開放プラットフォームで開発者登録済みであること

---

## ステップ 1: 開発者申請と Bot 作成

1. [QQ 開放プラットフォーム](https://q.qq.com) にアクセスする
2. **今すぐ接続** → 個人または企業として開発者申請を行う
3. 審査通過後、**ボット管理** → **ボット作成** を開く
4. Bot の基本情報を入力して作成する

---

## ステップ 2: AppID と AppSecret を取得する

1. ボット管理ページを開く
2. **開発設定** へ
3. **AppID** と **AppSecret** をコピーする

---

## ステップ 3: サンドボックスのテスト設定

正式リリース前に、サンドボックス環境でテストできます。

1. ボット管理コンソールで **サンドボックス設定** を探す
2. **メッセージリスト設定** で **メンバーを追加** をクリックし、自分の QQ 番号を入力する
3. 追加後、スマホ QQ で Bot の QR コードをスキャンする
4. Bot のホームを開き、**メッセージを送る** を押してテストを開始する

!!! warning "サンドボックスと本番"
    サンドボックスは開発テスト用です。本番公開にはコンソールで審査を提出して公開する必要があります。詳細は [QQ Bot ドキュメント](https://bot.q.qq.com/wiki/) を参照してください。

---

## ステップ 4: config.json を設定する

```json
{
  "channels": {
    "qq": {
      "enabled": true,
      "appId": "YOUR_APP_ID",
      "secret": "YOUR_APP_SECRET",
      "allowFrom": ["YOUR_OPENID"],
      "msgFormat": "plain"
    }
  }
}
```

### 完全な設定オプション

```json
{
  "channels": {
    "qq": {
      "enabled": true,
      "appId": "YOUR_APP_ID",
      "secret": "YOUR_APP_SECRET",
      "allowFrom": ["YOUR_OPENID"],
      "msgFormat": "plain"
    }
  }
}
```

| パラメータ | デフォルト | 説明 |
|------|--------|------|
| `enabled` | `false` | このチャンネルを有効にするか |
| `appId` | `""` | QQ Bot の AppID |
| `secret` | `""` | QQ Bot の AppSecret |
| `allowFrom` | `[]` | 対話を許可するユーザー OpenID のリスト |
| `msgFormat` | `"plain"` | メッセージ形式：`"plain"` または `"markdown"` |

### `msgFormat` の説明

| 値 | 適用シーン |
|----|----------|
| `"plain"`（デフォルト） | プレーンテキスト。すべての QQ クライアントと互換 |
| `"markdown"` | Markdown 形式。比較的新しい QQ クライアントのみ対応 |

---

## ステップ 5: 起動する

```bash
nanobot gateway
```

---

## 自分の OpenID を取得する

`allowFrom` に入れるのは QQ 番号ではなく、QQ のユーザー OpenID です。

**取得方法:**

1. まず `allowFrom` を `["*"]` にして全員許可にする
2. nanobot を起動して Bot にメッセージを送る
3. nanobot のログに OpenID が表示される
4. `allowFrom` を更新する

---

## グループ対応

現時点の QQ Bot 対応範囲:

- **1:1（C2C）**：一対一の私信
- **グループ @メンション**：グループ内で @ボット
- **Direct Message**：Guild 経由の DM

---

## 正式リリース手順

サンドボックステストが終わったら:

1. ボット管理コンソール → **バージョン管理** → 新しいバージョンを作成
2. 機能説明とスクリーンショットを入力
3. 審査を提出
4. 審査通過後に公開

詳細は [QQ Bot 公式ドキュメント](https://bot.q.qq.com/wiki/) を参照してください。

## よくある質問

**メッセージは届くのに返信しない？**

- `allowFrom` に自分の OpenID（QQ 番号ではない）が入っているか確認する
- ログに `Access denied` が出ていないか確認する

**グループで返信しない？**

- グループでは @ボットが必要です
- Bot がグループに追加されているか確認する

**サンドボックスで Bot が見つからない？**

- サンドボックス設定で自分の QQ 番号を追加したか確認する
- スマホ QQ（PC 版ではなく）で QR コードをスキャンする
