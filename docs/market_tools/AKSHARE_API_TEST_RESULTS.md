## ❌ 不可用的API接口

### 1. 美股实时行情 - `stock_us_spot()`
- **状态**: ❌ 失败
- **错误信息**: `Expecting value: line 1 column 1 (char 0)`
- **原因分析**: JSON解析失败，可能是API响应格式变化或网络问题
- **影响**: 无法获取美股实时数据
- **建议**: 
  - 尝试其他美股数据源（如yfinance）
  - 检查akshare版本是否需要更新

### 2. 国内期货 - `futures_zh_spot()`
- **状态**: ❌ 失败
- **错误信息**: `Length mismatch: Expected axis has 1 elements, new...`
- **原因分析**: 数据列数不匹配，API返回格式可能已变更
- **影响**: 无法获取期货实时数据
- **建议**: 检查akshare是否有更新的期货API

### 3. 可转债 - `bond_cb_spot_jsl()`
- **状态**: ❌ 失败
- **错误信息**: `AttributeError: module 'akshare' has no attribute 'bond_cb_spot_jsl'`
- **原因分析**: 该函数在当前akshare版本中不存在或已更名
- **影响**: 无法使用该API
- **建议**: 
  - 查阅akshare文档找到正确的可转债API
  - 可能的替代函数：`bond_cb_jsl()` 或其他类似函数

### 4. 宏观经济数据
- **GDP数据** - `macro_china_gdp_yearly()`: 未测试完成
- **CPI数据** - `macro_china_cpi()`: 未测试完成
- **原因**: 测试脚本在债券API处中断
