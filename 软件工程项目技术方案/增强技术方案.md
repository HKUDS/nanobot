# Nanobot日程规划模块 - 轻量级集成方案

基于您选择的方向，本方案采用轻量级集成策略，在现有Nanobot框架基础上扩展日程规划功能。

---

## 一、方案概述

### 1.1 设计理念
- **轻量集成**：不改变现有架构，作为技能集扩展
- **自然语言优先**：保留强大的自然语言解析能力
- **JSON数据交互**：前端通过JSON与后端通信
- **记忆增强**：利用现有MEMORY.md存储用户偏好

### 1.2 核心功能
1. **自然语言任务解析** - 将用户输入转化为结构化任务
2. **智能日程安排** - 基于优先级和时间的自动排程
3. **记忆增强存储** - 在MEMORY.md中存储用户习惯和偏好
4. **简单提醒系统** - 基于现有消息机制的提醒功能

---

## 二、架构设计

### 2.1 系统架构
```
用户界面 (前端)
    ↓ (JSON API)
Nanobot BFF服务
    ↓ (工具调用)
日程规划技能集 (skills/planner/)
    ↓ (数据存储)
MEMORY.md + 日程文件 (workspace/schedules/)
```

### 2.2 数据流设计
```
用户自然语言输入 → 前端 → BFF服务 → LLM解析 → 结构化任务 → 智能排程 → 存储 → 提醒
```

## 三、详细技术实现

### 3.1 核心数据模型

#### 任务数据结构
```python
class Task:
    id: str                    # 任务唯一标识
    name: str                  # 任务名称
    description: str           # 任务描述
    duration_minutes: int      # 预计耗时（分钟）
    priority: str              # 优先级：urgent_important/important/urgent/normal/low
    deadline: Optional[str]    # 截止时间（ISO格式）
    status: str                # 状态：pending/in_progress/completed/cancelled
    tags: List[str]            # 标签分类
    created_at: str            # 创建时间
    updated_at: str            # 更新时间
```

#### 日程数据结构
```python
class Schedule:
    date: str                  # 日期（YYYY-MM-DD）
    tasks: List[Task]          # 当日任务列表
    total_duration: int        # 总耗时
    available_time: int        # 可用时间（分钟）
    efficiency_score: float    # 日程效率评分
```

#### MEMORY.md增强格式
```markdown
# 用户偏好与习惯

## 日程偏好

## 长期目标
- 学习目标: 掌握Python高级特性
- 健康目标: 每周运动3次

## 习惯记录
```

### 3.4 前端交互设计

#### 自然语言输入界面
```javascript
// 用户输入组件
<ScheduleInput 
    onParse={(input) => {
        // 调用后端解析接口
        fetch(`/conversations/${conversationId}/schedule/parse`, {
            method: 'POST',
            body: JSON.stringify({user_input: input})
        })
        .then(response => response.json())
        .then(data => {
            // 显示解析结果和日程建议
            setParsedTasks(data.parsed_tasks);
            setScheduleSuggestions(data.schedule_suggestions);
        });
    }}
/>
```

#### 日程可视化界面
```javascript
// 日程展示组件
<ScheduleView 
    schedule={scheduleData}
    onTaskUpdate={(taskId, updates) => {
        // 更新任务状态
        fetch(`/conversations/${conversationId}/schedule/task/${taskId}`, {
            method: 'PATCH',
            body: JSON.stringify(updates)
        });
    }}
    onSetReminder={(taskId, reminderTime) => {
        // 设置提醒
        fetch(`/conversations/${conversationId}/schedule/reminder`, {
            method: 'POST',
            body: JSON.stringify({
                task_id: taskId,
                reminder_time: reminderTime
            })
        });
    }}
/>
```
