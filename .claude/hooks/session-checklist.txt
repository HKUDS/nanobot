- New code MUST follow package conventions, not plan code examples

During implementation:

- Clear .mypy_cache after deleting files

Before completing:
- NEVER skip code quality review — every critical finding in the 2026-03-28 audit would have been caught by a reviewer
- If deviating from spec: update the spec in the same commit
- Run make pre-push (not just make check) before pushing
