# TRANS-LOCAL: Локальная транскрибация голосовых сообщений

Нужно чтобы у пользователя была возможность установить локально необходимые для транскрибации инструменты и настроить нейроассистента для использования локальной транскрибации.

Нужно:
- создать програмный интерфейс, который будет выбирать настроенный транскрибатор
- выбор транскрибатора по умолчанию стоит в auto, назначить его вручную можно в конфиге agents -> defaults
- настройки транскрибаторов в отдельной ветке конфигурации
- для локальной транскрибации используется faster wisper
