## 技术架构

### ContextVar会话级配置
```python
# 每个实验设置独立的会话配置
session_config = {
    "session_key": "DeepSeek_Task1_rep1",
    "api_type": "deepseek",
    "model": "deepseek-chat",
    "temperature": 0.7,
    "max_tokens": 4096
}

# 设置到ContextVar（线程安全、协程隔离）
set_current_session_config(session_config)

# API调用时自动读取会话配置
session_config = get_current_session_config()
api_type = session_config["api_type"]  # 自动路由到正确的API
```

### 架构优势
1. **零上下文污染**: 每个会话独立配置，互不干扰
2. **高性能**: ContextVar操作开销极小（~0.1μs）
3. **异步安全**: 支持高并发实验执行
4. **向后兼容**: 不影响现有代码
