# 基于悬赏机制与条件反射的多智能体协作进化系统技术方案

## 一、核心概念概述

本方案在现有 Nanobot 容器化智能体管理系统基础上，引入**分布式协作网络**与**生物启发的学习机制**，实现：

- **悬赏发布**：Agent（或用户）遇到无法解决的难题时，可发起悬赏，向好友或全网广播求助。
- **协作求解**：多个 Agent 可消耗 Token 参与悬赏，提交方案（Skill 或文档）。
- **评审与奖励**：发起者评估结果，对前三名贡献者分配 Token 奖励，并将最优方案沉淀为公共 Skill 或知识文档。
- **条件反射学习**：Agent 对常见问题建立“刺激-反应”快速通道（基础反射），并通过经验积累形成新的条件反射（类比巴甫洛夫实验）。

## 二、系统架构扩展

### 2.1 新增组件

| 组件 | 功能 |
|------|------|
| **悬赏中心 (BountyHub)** | 管理悬赏任务的生命周期（发布、参与、评审、结题）。 |
| **Token 钱包 (TokenWallet)** | 记录每个 Agent（对话容器）持有的 Token 数量，支持转账、消耗、奖励。 |
| **公共知识库 (PublicSpace)** | 存储沉淀的 Skill 定义、解决方案文档、最佳实践。支持语义检索。 |
| **反射库 (ReflexLibrary)** | 存储“条件反射”规则：输入模式 → 推荐行动/技能。支持快速匹配。 |
| **社交记忆 (SocialMemory)** | 记录 Agent 间的“好友”关系、历史协作记录、信任度评分。 |

### 3.2 Token 经济模型

- **初始 Token**：每个新容器创建时获得初始 Token（如 1000）。
- **消耗场景**：
  - 发起悬赏：扣除奖励池总额（从发起者钱包转出）。
  - 参与悬赏：扣除 `cost_tokens`（从参与者钱包转出，若未获胜则丢失）。
  - 查询公共知识库：每次检索消耗少量 Token（可选）。
- **获得 Token**：
  - 完成对话获得基础 Power 奖励（自动）。
  - 悬赏获胜：获得奖励池中的份额。
  - 贡献优质 Skill 到公共库：获得 Token 奖励（由系统或社区投票）。

### 3.3 公共知识库 (PublicSpace)

- **存储内容**：
  - Skill 定义（可执行代码 + 元数据）。
  - 解决方案文档（Markdown）。
  - 问题-答案对（FAQ）。
- **检索方式**：
  - 关键词搜索。
  - 语义检索（ChromaDB），将问题描述向量化，匹配相似历史问题。
- **贡献奖励**：当 Agent 提交的 Skill 被采纳（被其他 Agent 使用超过阈值），给予 Token 奖励。

### 3.4 条件反射机制

#### 3.4.1 基础反射（先天）

- 预定义一组简单模式匹配规则，例如：
  - 用户输入包含“天气” → 调用 `weather_search` 工具。
  - 用户输入包含“计算” → 调用 `calculator` 工具。
- 这些规则存储在 `ReflexLibrary` 的 `innate` 表中，优先级高，无需学习。

#### 3.4.2 条件反射（后天）

- **形成过程**：
  1. Agent 多次成功解决某类问题（高 reward 轨迹）。
  2. 系统提取该问题的**特征向量**（来自 `state.goal`, `observation.content` 等）和采取的**动作序列**。
  3. 将“特征 → 动作”存储为一条条件反射规则，并记录置信度。
  4. 后续遇到相似问题时，优先匹配条件反射，跳过复杂推理，直接执行预设动作。

- **数据结构**：

```python
class ConditionedReflex(BaseModel):
    id: str
    feature_vector: List[float]   # 向量化的问题特征
    action_sequence: List[dict]    # 动作列表（工具调用/回复模板）
    confidence: float              # 置信度（基于成功次数）
    source_agent_id: str
    usage_count: int
```

- **匹配与执行**：
  - 新消息进入后，先计算特征向量，与 `ReflexLibrary` 中的规则进行相似度计算（余弦相似度）。
  - 相似度超过阈值（如 0.85）且置信度足够高，则直接执行预定义动作，无需调用 LLM。
  - 执行结果若成功，增加规则的置信度和使用次数；若失败，降低置信度或标记为待验证。

#### 3.4.3 反射的共享与进化

- Agent 可将自己形成的条件反射规则上传到公共空间，供其他 Agent 使用。
- 上传者可获得 Token 奖励（按使用次数分成）。
- 规则失效后可被淘汰（使用次数少、置信度低）。

### 3.5 社交记忆与好友机制

- **好友关系**：Agent 可主动添加其他 Agent 为好友（基于信任度或历史协作）。
- **选择性求助**：发起悬赏时，可选择“仅好友可见”或“全网公开”。
- **信任度评分**：根据历史协作成功率、奖励公正性等因素，动态计算信任分。
- **好友推荐**：当 Agent 遇到难题时，系统自动推荐信任度高且擅长相关领域的“好友”。

## 四、与现有系统的集成

## 五、工作流程示例

1. **AgentA 遇到难题**：用户发送复杂任务“分析这份财报并生成摘要”。AgentA 内部推理失败（多次工具调用错误，reward 低）。
2. **发起悬赏**：AgentA 自动（或用户手动）发布悬赏：“需财务分析能力，奖励 500 Token”。
3. **广播与参与**：
   - AgentB（好友）看到悬赏，消耗 10 Token 参与，提交解决方案（含分析脚本）。
   - AgentC（陌生人）也参与。
4. **评审**：AgentA 执行所有方案，根据效果评分（可调用 LLM 辅助评分）。
5. **奖励与沉淀**：AgentA 将 500 Token 分配给前三名（如 300, 150, 50）。最优方案被转化为公共 Skill，上传到知识库。
6. **条件反射**：AgentA 从成功方案中学习到“财务分析 → 调用 `financial_analyzer` 工具”，形成条件反射。后续类似问题直接触发该反射，无需再悬赏。

## 六、技术挑战与应对

| 挑战 | 解决方案 |
|------|----------|
| Token 双花攻击 | 使用乐观锁或数据库事务保证原子性 |
| 悬赏任务重复提交 | 基于问题哈希去重，或限制单个 Agent 同时参与数量 |
| 反射规则质量评估 | 引入“置信度”衰减机制，长期未使用的规则降低权重 |
| 知识库爆炸 | 定期清理低质量内容，采用投票/点赞机制 |
| 跨容器通信 | 使用 BFF 作为中央枢纽，通过 HTTP/WebSocket 转发消息 |

import sqlite3
from pathlib import Path

DB_PATH = Path(__file__).parent / "bff.db"

def init_db():
    with sqlite3.connect(DB_PATH) as conn:

            CREATE TABLE IF NOT EXISTS bounties (

                created_at TIMESTAMP,

                skill_code TEXT,
                cost_tokens INTEGER DEFAULT 0,
                evaluation_score REAL,
                created_at TIMESTAMP,
                FOREIGN KEY (bounty_id) REFERENCES bounties(id),
                FOREIGN KEY (agent_id) REFERENCES wallets(conversation_id)
            )
        """)

            CREATE TABLE IF NOT EXISTS public_knowledge (
                id TEXT PRIMARY KEY,
                type TEXT CHECK(type IN ('skill', 'doc', 'faq')),

                content TEXT,
                skill_code TEXT,
                tags TEXT,
                author_id TEXT,
                usage_count INTEGER DEFAULT 0,
                token_reward INTEGER DEFAULT 0,
                created_at TIMESTAMP

        """)

            CREATE TABLE IF NOT EXISTS reflexes (

                action_sequence TEXT,
                confidence REAL DEFAULT 0.5,
                source_agent_id TEXT,
                usage_count INTEGER DEFAULT 0,
                success_count INTEGER DEFAULT 0,

        """)

            CREATE TABLE IF NOT EXISTS friendships (

## 二、Token 钱包模块

### 2.1 新增文件 `bff/token_wallet.py`

```python
import sqlite3
import asyncio
from datetime import datetime
from bff.db import DB_PATH

class TokenWallet:
    def __init__(self):
        self._lock = asyncio.Lock()

    async def get_balance(self, conv_id: str) -> int:
        with sqlite3.connect(DB_PATH) as conn:
            cur = conn.execute("SELECT balance FROM wallets WHERE conversation_id = ?", (conv_id,))
            row = cur.fetchone()
            if not row:
                # 初始化钱包
                conn.execute("INSERT INTO wallets (conversation_id, balance, updated_at) VALUES (?, 1000, ?)",
                             (conv_id, datetime.now()))
                return 1000
            return row[0]

    async def transfer(self, from_id: str, to_id: str, amount: int, reason: str, bounty_id: str = None):
        if amount <= 0:
            raise ValueError("Amount must be positive")
        async with self._lock:
            with sqlite3.connect(DB_PATH) as conn:
                # 检查余额
                cur = conn.execute("SELECT balance FROM wallets WHERE conversation_id = ?", (from_id,))
                row = cur.fetchone()
                if not row or row[0] < amount:
                    raise ValueError("Insufficient balance")
                # 扣款
                conn.execute("UPDATE wallets SET balance = balance - ?, updated_at = ? WHERE conversation_id = ?",
                             (amount, datetime.now(), from_id))
                # 加款
                conn.execute("UPDATE wallets SET balance = balance + ?, updated_at = ? WHERE conversation_id = ?",
                             (amount, datetime.now(), to_id))
                # 记录交易
                conn.execute("""
                    INSERT INTO transactions (from_id, to_id, amount, reason, bounty_id, created_at)
                    VALUES (?, ?, ?, ?, ?, ?)
                """, (from_id, to_id, amount, reason, bounty_id, datetime.now()))

    async def ensure_wallet(self, conv_id: str):
        """确保钱包存在"""
        with sqlite3.connect(DB_PATH) as conn:
            conn.execute("INSERT OR IGNORE INTO wallets (conversation_id, balance, updated_at) VALUES (?, 1000, ?)",
                         (conv_id, datetime.now()))
```

## 三、悬赏中心模块

### 3.1 新增文件 `bff/bounty_hub.py`

```python
import uuid
import sqlite3
from datetime import datetime
from typing import List, Optional
from pydantic import BaseModel
from bff.db import DB_PATH
from bff.token_wallet import TokenWallet

class SubmissionCreate(BaseModel):
    content: str
    skill_code: Optional[str] = None
    cost_tokens: int = 0

class BountyCreate(BaseModel):
    title: str
    description: str
    reward_pool: int
    deadline: datetime

class BountyHub:
    def __init__(self, wallet: TokenWallet):
        self.wallet = wallet

    async def create_bounty(self, issuer_id: str, bounty: BountyCreate) -> str:
        # 扣除奖励池 Token（转入系统暂存，这里简化为直接扣减，后续奖励时转出）
        await self.wallet.transfer(issuer_id, "system", bounty.reward_pool, "bounty_lock")
        bounty_id = str(uuid.uuid4())
        with sqlite3.connect(DB_PATH) as conn:
            conn.execute("""
                INSERT INTO bounties (id, issuer_id, title, description, reward_pool, deadline, status, created_at)
                VALUES (?, ?, ?, ?, ?, ?, 'open', ?)
            """, (bounty_id, issuer_id, bounty.title, bounty.description, bounty.reward_pool, bounty.deadline, datetime.now()))
        return bounty_id

    async def submit_solution(self, bounty_id: str, agent_id: str, submission: SubmissionCreate) -> str:
        # 扣除参与费用（如果 cost_tokens > 0）
        if submission.cost_tokens > 0:
            await self.wallet.transfer(agent_id, "system", submission.cost_tokens, "bounty_participation", bounty_id)
        sub_id = str(uuid.uuid4())
        with sqlite3.connect(DB_PATH) as conn:
            conn.execute("""
                INSERT INTO submissions (id, bounty_id, agent_id, content, skill_code, cost_tokens, created_at)
                VALUES (?, ?, ?, ?, ?, ?, ?)
            """, (sub_id, bounty_id, agent_id, submission.content, submission.skill_code, submission.cost_tokens, datetime.now()))
        return sub_id

    async def evaluate_and_reward(self, bounty_id: str, winner_submission_ids: List[str], scores: List[float]):
        # 获取悬赏信息
        with sqlite3.connect(DB_PATH) as conn:
            row = conn.execute("SELECT reward_pool FROM bounties WHERE id = ?", (bounty_id,)).fetchone()
            if not row:
                raise ValueError("Bounty not found")
            reward_pool = row[0]
        # 按比例分配奖励（第一名50%，第二名30%，第三名20%）
        ratios = [0.5, 0.3, 0.2]
        for i, sub_id in enumerate(winner_submission_ids):
            if i >= len(ratios):
                break
            amount = int(reward_pool * ratios[i])
            # 获取提交者 agent_id
            with sqlite3.connect(DB_PATH) as conn:
                agent_row = conn.execute("SELECT agent_id FROM submissions WHERE id = ?", (sub_id,)).fetchone()
                if not agent_row:
                    continue
                agent_id = agent_row[0]
            # 转账奖励（从 system 转到 agent）
            await self.wallet.transfer("system", agent_id, amount, "bounty_reward", bounty_id)
            # 更新 submission 的 evaluation_score
            with sqlite3.connect(DB_PATH) as conn:
                conn.execute("UPDATE submissions SET evaluation_score = ? WHERE id = ?", (scores[i], sub_id))
        # 更新悬赏状态
        with sqlite3.connect(DB_PATH) as conn:
            conn.execute("UPDATE bounties SET status = 'closed', winner_ids = ? WHERE id = ?",
                         (",".join(winner_submission_ids), bounty_id))

    async def list_open_bounties(self) -> List[dict]:
        with sqlite3.connect(DB_PATH) as conn:
            conn.row_factory = sqlite3.Row
            rows = conn.execute("SELECT * FROM bounties WHERE status = 'open' AND deadline > ?", (datetime.now(),)).fetchall()
            return [dict(row) for row in rows]
```

```python
import uuid
import sqlite3
import chromadb
from chromadb.utils import embedding_functions
from bff.db import DB_PATH

class PublicSpace:
    def __init__(self, chroma_path: str = "./chroma_data"):
        self.sql_conn = sqlite3.connect(DB_PATH, check_same_thread=False)
        self.chroma_client = chromadb.PersistentClient(path=chroma_path)
        self.embedding_fn = embedding_functions.SentenceTransformerEmbeddingFunction(model_name="all-MiniLM-L6-v2")
        self.collection = self.chroma_client.get_or_create_collection(
            name="public_knowledge",
            embedding_function=self.embedding_fn
        )

    async def search(self, query: str, top_k: int = 5) -> List[dict]:
        results = self.collection.query(query_texts=[query], n_results=top_k)
        # 从 SQLite 获取完整信息
        ids = results['ids'][0]
        docs = []
        with sqlite3.connect(DB_PATH) as conn:
            for doc_id in ids:
                row = conn.execute("SELECT * FROM public_knowledge WHERE id = ?", (doc_id,)).fetchone()
                if row:
                    docs.append(dict(zip([c[0] for c in conn.description], row)))
        return docs

    async def upload(self, title: str, content: str, skill_code: str, tags: list, author_id: str, knowledge_type: str = "skill"):
        doc_id = str(uuid.uuid4())
        tags_json = json.dumps(tags)
        with sqlite3.connect(DB_PATH) as conn:
            conn.execute("""
                INSERT INTO public_knowledge (id, type, title, content, skill_code, tags, author_id, created_at)
                VALUES (?, ?, ?, ?, ?, ?, ?, ?)
            """, (doc_id, knowledge_type, title, content, skill_code, tags_json, author_id, datetime.now()))
        # 向量化文本
        text_to_embed = f"{title}\n{content}\n{skill_code if skill_code else ''}"
        self.collection.add(
            ids=[doc_id],
            documents=[text_to_embed],
            metadatas=[{"title": title, "tags": tags, "author": author_id}]
        )
        return doc_id
```

## 五、条件反射引擎

### 5.1 新增文件 `bff/reflex_engine.py`

```python
import sqlite3
import json
import numpy as np
from sentence_transformers import SentenceTransformer
from bff.db import DB_PATH

class ReflexEngine:
    def __init__(self):
        self.model = SentenceTransformer('all-MiniLM-L6-v2')
        self._init_reflex_table()

    def _init_reflex_table(self):
        with sqlite3.connect(DB_PATH) as conn:
            conn.execute("""
                CREATE TABLE IF NOT EXISTS reflexes (
                    id TEXT PRIMARY KEY,
                    feature_text TEXT,
                    action_sequence TEXT,
                    confidence REAL DEFAULT 0.5,
                    source_agent_id TEXT,
                    usage_count INTEGER DEFAULT 0,
                    success_count INTEGER DEFAULT 0,
                    created_at TIMESTAMP
                )
            """)

    def _embed(self, text: str) -> np.ndarray:
        return self.model.encode(text)

    async def match(self, state: dict, threshold: float = 0.85) -> Optional[dict]:
        """从 state 中提取特征文本，匹配最相似的条件反射"""
        feature_text = self._extract_feature_text(state)
        query_vec = self._embed(feature_text)
        with sqlite3.connect(DB_PATH) as conn:
            conn.row_factory = sqlite3.Row
            rows = conn.execute("SELECT id, feature_text, action_sequence, confidence FROM reflexes WHERE confidence > ?", (0.3,)).fetchall()
        best = None
        best_score = 0
        for row in rows:
            stored_vec = self._embed(row["feature_text"])
            sim = np.dot(query_vec, stored_vec) / (np.linalg.norm(query_vec) * np.linalg.norm(stored_vec))
            if sim > threshold and sim > best_score:
                best_score = sim
                best = dict(row)
        if best:
            return {"action_sequence": json.loads(best["action_sequence"]), "confidence": best["confidence"]}
        return None

    def _extract_feature_text(self, state: dict) -> str:
        goal = state.get("goal", "")
        history = state.get("history_summary", "")
        skills = ", ".join(state.get("available_skills", []))
        env = state.get("environment", {}).get("type", "")
        return f"{goal} {history} {skills} {env}"

    async def learn(self, state: dict, action_sequence: list, agent_id: str, success: bool = True):
        """从成功的对话中学习条件反射"""
        feature_text = self._extract_feature_text(state)
        action_json = json.dumps(action_sequence)
        with sqlite3.connect(DB_PATH) as conn:
            # 检查是否已存在相似规则（特征文本完全相同）
            cur = conn.execute("SELECT id, confidence, usage_count, success_count FROM reflexes WHERE feature_text = ?", (feature_text,))
            row = cur.fetchone()
            if row:
                # 更新已有规则
                new_usage = row[2] + 1
                new_success = row[3] + (1 if success else 0)
                new_conf = row[1] + 0.05 if success else row[1] - 0.05
                new_conf = max(0.1, min(1.0, new_conf))
                conn.execute("""
                    UPDATE reflexes SET confidence = ?, usage_count = ?, success_count = ? WHERE id = ?
                """, (new_conf, new_usage, new_success, row[0]))
            else:
                # 新增规则
                import uuid
                rid = str(uuid.uuid4())
                conn.execute("""
                    INSERT INTO reflexes (id, feature_text, action_sequence, confidence, source_agent_id, usage_count, success_count, created_at)
                    VALUES (?, ?, ?, ?, ?, ?, ?, ?)
                """, (rid, feature_text, action_json, 0.5, agent_id, 1, 1 if success else 0, datetime.now()))
```

## 六、社交记忆与好友机制

### 6.1 新增文件 `bff/social_graph.py`

```python
import sqlite3
from bff.db import DB_PATH

class SocialGraph:
    async def add_friend(self, agent_a: str, agent_b: str):
        with sqlite3.connect(DB_PATH) as conn:
            conn.execute("INSERT OR IGNORE INTO friendships (agent_a, agent_b, trust_score, created_at) VALUES (?, ?, 0.5, ?)", (agent_a, agent_b, datetime.now()))
            conn.execute("INSERT OR IGNORE INTO friendships (agent_a, agent_b, trust_score, created_at) VALUES (?, ?, 0.5, ?)", (agent_b, agent_a, datetime.now()))

    async def update_trust(self, agent_a: str, agent_b: str, delta: float):
        with sqlite3.connect(DB_PATH) as conn:
            conn.execute("UPDATE friendships SET trust_score = MIN(1.0, MAX(0.0, trust_score + ?)) WHERE agent_a = ? AND agent_b = ?", (delta, agent_a, agent_b))

    async def get_friends(self, agent_id: str) -> List[str]:
        with sqlite3.connect(DB_PATH) as conn:
            rows = conn.execute("SELECT agent_b FROM friendships WHERE agent_a = ? AND trust_score > 0.3", (agent_id,)).fetchall()
            return [r[0] for r in rows]

    async def recommend_friends(self, agent_id: str, task_description: str, top_k: int = 3) -> List[str]:
        # 简单实现：返回信任度最高的好友
        with sqlite3.connect(DB_PATH) as conn:
            rows = conn.execute("SELECT agent_b, trust_score FROM friendships WHERE agent_a = ? ORDER BY trust_score DESC LIMIT ?", (agent_id, top_k)).fetchall()
            return [r[0] for r in rows]
```

### 8.1 新增工具函数

在 `agent_server.py` 的 `initialize_agent` 之后，注册自定义工具：

```python
from nanobot.agent.tools.base import Tool

class PublishBountyTool(Tool):
    name = "publish_bounty"
    description = "发布悬赏任务，向其他Agent求助"
    parameters = {
        "type": "object",
        "properties": {
            "title": {"type": "string"},
            "description": {"type": "string"},
            "reward_pool": {"type": "integer"},
            "deadline": {"type": "string"}
        }
    }
    async def call(self, title: str, description: str, reward_pool: int, deadline: str):
        # 调用 BFF 接口
        async with aiohttp.ClientSession() as session:
            async with session.post("http://bff:8000/bounties", json={
                "title": title,
                "description": description,
                "reward_pool": reward_pool,
                "deadline": deadline
            }) as resp:
                return await resp.text()
```

然后在 `AgentLoop` 创建后，将工具注册到 `tool_registry`。

由于 `agent_server.py` 中已经创建了 `agent_loop`，可以通过 `agent_loop.tools.register(PublishBountyTool())` 添加。
