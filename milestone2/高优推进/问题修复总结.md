# 悬赏任务系统问题总结与修复

## 一、问题列表

### 1. 容器 CONVERSATION_ID=unknown
- **现象**：容器日志显示 `conversation=unknown`
- **原因**：`fork_container` 方法中未正确传递 `CONVERSATION_ID`
- **修复**：
  - 在 `allowed_prefixes` 中添加 `CONVERSATION_ID`
  - 显式设置子容器的新 `CONVERSATION_ID`

### 2. 变量覆盖导致提交失败
- **现象**：`'Session' object has no attribute 'post'`
- **原因**：`session` 变量被 AgentLoop 的会话对象覆盖
- **修复**：将 `aiohttp.ClientSession` 变量改为 `http_session`

### 3. Nanobot Consolidator TypeError
- **现象**：`TypeError: '<=' not supported between instances of 'NoneType' and 'int'`
- **原因**：`context_window_tokens` 为 None，与整数比较失败
- **修复**：
  - Monkey Patch `Consolidator.maybe_consolidate_by_tokens`
  - 使用 `*args, **kwargs` 捕获所有参数

### 4. Solution 内容提取失败
- **现象**：提交内容为占位文本而非实际生成内容
- **原因**：
  - `msg['role']` 可能不是 `'assistant'`
  - `response.content` 可能为空
  - 会话消息未及时更新
- **修复**：
  - 扩展角色匹配：`{'assistant', 'model', 'ai', 'bot'}`
  - 添加详细调试日志
  - 多级备选提取：session → response.content → response.text

### 5. 通知延迟问题
- **现象**：发布悬赏后邻居不能立即收到通知
- **原因**：轮询间隔过长（30秒）
- **修复**：
  - 容器端：改为 5 秒轮询
  - 前端：改为 5 秒轮询

### 6. 邻居发现逻辑错误（全连接问题）
- **现象**：所有节点都能收到通知，而非只有邻居
- **原因**：`_discover_and_connect_neighbors` 做了全连接
- **修复**：改为只建立父子节点的邻居关系

### 7. API KEY 未正确传递
- **现象**：`[Evaluator] 未配置 DEEPSEEK_API_KEY，使用规则评分`
- **修复**：
  - `evaluator.py` 使用 `shared/config.py` 中的硬编码 API KEY
  - `bff_service.py` 中 `api_ai_assist` 使用硬编码值

### 8. 重复提交问题
- **现象**：一个节点接受后所有节点都 processing
- **修复**：在 `submit_solution` 中添加重复检查

## 五、技术细节

### 5.1 Monkey Patch 代码
```python
from nanobot.agent.memory import Consolidator
_original_maybe = Consolidator.maybe_consolidate_by_tokens

async def _patched_maybe(self, *args, **kwargs):
    if hasattr(self, 'context_window_tokens') and self.context_window_tokens is None:
        self.context_window_tokens = 128000
    return await _original_maybe(self, *args, **kwargs)

Consolidator.maybe_consolidate_by_tokens = _patched_maybe
```

### 5.2 Solution 提取逻辑
```python
assistant_roles = {'assistant', 'model', 'ai', 'bot'}
for msg in reversed(messages):
    role = msg.get('role', '').lower()
    content = msg.get('content', '')
    if role in assistant_roles and content and content.strip():
        solution_content = content.strip()
        break
```
