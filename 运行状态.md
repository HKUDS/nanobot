### 常见问题

**Q: 机器人收到消息但不回复？**
A: 检查 API 密钥是否有效，运行 `nanobot agent -m "测试"` 测试 LLM 连接。

**Q: WebSocket 连接断开？**
A: 这是正常的，nanobot 会自动重连。如果持续断开，检查网络连接。

**Q: 如何更改模型？**
A: 编辑 `~/.nanobot/config.json`，修改 `agents.defaults.model`，然后重启 gateway。
