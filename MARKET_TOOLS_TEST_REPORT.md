## API变更说明

### 原方案：东方财富API ❌
- 实时行情：`stock_zh_a_spot_em()`
- 历史数据：`stock_zh_a_hist()`
- **问题**：代理连接错误，无法访问 `82.push2.eastmoney.com` 和 `push2his.eastmoney.com`

### 新方案：新浪财经API ✅
- 实时行情：`stock_zh_a_spot()`
- 港股实时：`stock_hk_spot()`
- 美股实时：`stock_us_spot()`
- **优势**：
  - 更稳定，不需要特殊代理配置
  - 返回中文字段名，与原有格式化代码兼容
  - 支持A股、港股、美股全市场数据

## 代码改进

### 修改的文件
- `nanobot/agent/tools/market.py`

### 主要改动

1. **实时行情API切换**
   ```python
   # 从东方财富改为新浪财经
   df = ak.stock_zh_a_spot()  # 原来是 stock_zh_a_spot_em()
   ```

2. **字段名兼容性**
   - 新浪财经API返回中文字段名（'代码', '名称', '最新价'等）
   - 与原有格式化函数完全兼容，无需修改

3. **代理处理优化**
   - 保留了 `_disable_proxy_for_akshare()` 函数
   - 虽然新浪API不严格要求禁用代理，但保留以防万一

## 使用示例

```python
import asyncio
from nanobot.agent.tools.market import StockPriceTool

async def main():
    tool = StockPriceTool()
    
    # 获取A股实时行情
    result = await tool.execute(
        symbol="bj920000",  # 股票代码
        market="cn",         # 市场类型：cn/hk/us
        period="realtime"    # 数据类型：realtime/daily/weekly/monthly
    )
    print(result)

if __name__ == "__main__":
    asyncio.run(main())
```
