# 市场分析框架

## 概述

市场分析是量化投资决策的基础。系统的市场分析框架可以帮助投资者理解当前市场环境，判断市场周期位置，从而做出更合理的投资决策。

## 宏观周期判断

### 美林投资时钟

根据经济增长（GDP）和通胀两个维度，将经济周期分为四个阶段：

| 阶段 | 经济 | 通胀 | 最佳资产 |
|------|------|------|----------|
| 复苏 | ↑ | ↓ | 股票 |
| 繁荣 | ↑ | ↑ | 商品 |
| 衰退 | ↓ | ↓ | 债券 |
| 萧条 | ↓ | ↑ | 现金 |

### 周期指标

**经济增长指标**：
- GDP增速
- PMI（采购经理人指数）
- 工业增加值
- 社融增速

**通胀指标**：
- CPI（居民消费价格指数）
- PPI（工业生产者出厂价格指数）
- M2增速

### A股特色周期

- **政策周期**：货币政策、财政政策、监管政策
- **库存周期**：被动去库→主动补库
- **信用周期**：宽信用→紧信用
- **产业周期**：技术革新驱动的产业发展

## 行业轮动模型

### 行业轮动逻辑

**经济复苏期**：金融、可选消费→周期成长
**经济繁荣期**：周期品、有色、煤炭→科技
**经济衰退期**：医药、公用事业→必选消费
**经济萧条期**：现金为王

### 动量轮动指标

| 指标 | 信号 | 含义 |
|------|------|------|
| 行业20日动量 | 强 | 趋势向上 |
| 行业波动率 | 高 | 风险加大 |
| 行业成交额占比 | 高 | 情绪热度高 |
| 北向资金净流入 | 持续 | 机构看好 |

### 行业配置框架

```
1. 判断宏观周期位置
   ↓
2. 确定超配行业方向
   ↓
3. 筛选行业内强势标的
   ↓
4. 设定仓位和止损
```

## 资金流向分析

### 主要资金来源

| 资金类型 | 特点 | 影响 |
|----------|------|------|
| 北向资金 | 机构为主，偏向价值 | 风向标 |
| 公募基金 | 抱团取暖 | 推动行情 |
| 私募基金 | 灵活 | 择时明显 |
| 社保资金 | 长线 | 稳定器 |
| 杠杆资金 | 追涨杀跌 | 放大波动 |

### 资金流向信号

- **北向资金持续净流入**：市场有望上涨
- **杠杆资金大幅增加**：风险积累
- **新基金发行火爆**：市场可能过热
- **融资余额持续下降**：市场情绪低迷

### 资金流向数据来源

- 东方财富网：每日资金流向
- 同花顺：行业/个股资金流向
- Wind：机构持仓数据

## 市场情绪指标

### 情绪度量指标

| 指标 | 低位阈值 | 高位阈值 | 含义 |
|------|----------|----------|------|
| 融资融券余额 | 1.4万亿 | 1.8万亿 | 杠杆水平 |
| 换手率 | 0.5% | 1.5% | 交易活跃度 |
| 涨停家数 | <20 | >100 | 赚钱效应 |
| 下跌家数/上涨家数 | <0.3 | >2 | 市场强弱 |

### 市场情绪周期

```
兴奋 → 乐观 → 平静 → 悲观 → 绝望
  ↑                              ↓
  ←←←←←←←←←←←←←←←←←←←←←←←←←←
```

### 情绪指标应用

- **极度悲观时**：可能是买入机会
- **极度乐观时**：风险积累，注意减仓
- **情绪极端但指数不极端**：警惕背离

## 股债性价比

### 概念

股债性价比 = 股票预期收益率 / 债券收益率

### 常用指标

**风险溢价**：
- E/P - 10年期国债收益率
- 股息率 - 10年期国债收益率

### 判断标准

| 风险溢价 | 股债比 | 信号 |
|----------|--------|------|
| >3% | >1.5 | 股票极端低估 |
| 1-3% | 1.0-1.5 | 股票相对占优 |
| 0-1% | 0.8-1.0 | 债券相对占优 |
| <0% | <0.8 | 股票极端高估 |

### 应用注意

- 极端值往往持续很长时间
- 需要结合基本面判断
- 单一指标不宜作为决策依据

## 市场分析框架总结

### 宏观层面
- 经济周期位置判断
- 政策导向分析
- 通胀和流动性环境

### 中观层面
- 行业轮动方向
- 景气行业筛选
- 资金配置方向

### 微观层面
- 市场情绪温度
- 资金流向跟踪
- 估值分位判断

### 决策流程

```
1. 宏观环境分析
   ├─ 经济周期
   ├─ 政策周期
   └─ 流动性环境

2. 市场状态判断
   ├─ 估值分位
   ├─ 情绪指标
   └─ 趋势状态

3. 行业选择
   ├─ 景气行业
   ├─ 轮动方向
   └─ 资金偏好

4. 个股/策略选择
   ├─ 因子配置
   └─ 仓位管理
```

## 风险提示

- 市场分析是概率游戏，没有100%准确的方法
- 单一指标可能失效，需要多维度验证
- 分析结论需要持续跟踪和调整
- 过去有效不代表未来有效
